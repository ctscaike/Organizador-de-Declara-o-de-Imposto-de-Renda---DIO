# Organizador de Declaração de Imposto de Renda

## Descrição

Este projeto consiste em uma planilha desenvolvida para organizar informações necessárias para a declaração do imposto de renda. A planilha contém três abas principais:

- **TITULAR:** Dados pessoais do titular da declaração.
- **INFORMES:** Informações sobre rendimentos bancários.
- **NOTAS:** Registro de entradas financeiras, como holerites.

## Objetivo

O objetivo principal foi acompanhar as aulas do curso disponibilizado, compreendendo o funcionamento e a aplicação prática por meio da planilha estudada.

## Observação Importante

Por questão crítica de tempo, optei por acompanhar as aulas assistindo aos vídeos e compreendendo o que foi realizado na planilha, sem desenvolver códigos adicionais ou outras implementações.
